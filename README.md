# 📊 Dashboard de Vendas

Dashboard interativo desenvolvido com Python e Streamlit
para visualização de dados de vendas.

## 🚀 Acesse o Dashboard Online

🔗 [Clique aqui para abrir o Dashboard](https://dashboard-vendas-ejefsl3xx4kcxbdm3wmcac.streamlit.app/)

## 🛠️ Tecnologias Utilizadas

- Python
- Streamlit
- Plotly
- Pandas

## 📁 Arquivos

| Arquivo | Descrição |
|---|---|
| `streamlit_app3.py` | Código principal do dashboard |
| `vendas_dashboard.csv` | Base de dados de vendas |
| `requirements.txt` | Dependências do projeto |

## 👨‍💻 Como Rodar Localmente

```bash
pip install -r requirements.txt
streamlit run streamlit_app3.py
```
